## MVP / promotion gate

The repository is eligible to be promoted to its final standalone location when:

1. all local checks above remain green on the tagged commit;
2. the GitHub Release workflow publishes all assets, checksums, and SPDX SBOM;
3. the public Release asset set passes package smoke and secret scan;
4. the final Release URL and commit are recorded in `STATUS.md`.

The future filesystem location is not a runtime contract. Source, workflows,
documentation, launchers, and profiles use repository-relative or configurable
paths.
