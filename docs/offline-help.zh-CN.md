# SSH Launchpad 离线帮助

## 最简单的使用方法

Windows 桌面用户优先下载安装器，双击安装后按中文向导操作。服务器或无桌面环境请完整解压 portable ZIP，再双击 `开始使用 SSH Launchpad.cmd`。

## 公钥不要弄反

- 正在设置的是“被连接电脑”。
- 需要提供的是“控制电脑”的 `.pub` 公钥。
- 私钥没有 `.pub` 后缀，必须留在控制电脑上；不要复制、粘贴、上传或放进 profile。
- 第一次从控制电脑连接时，要核对主机指纹，不要静默接受未知指纹。

## 没有网络

可以稍后重试、配置显式代理/HTTPS 镜像，或使用同目录带 `checksums.txt` 的离线包。校验失败的下载不会被执行。不要关闭 TLS 或系统安全软件来绕过错误。

## 管理员授权、取消和恢复

普通用户启动即可；只有实际安装时才请求 UAC/sudo。取消授权后不会继续执行。发生部分失败时，工具会停止后续步骤并按执行记录恢复可逆改动；高级模式可导出报告或选择“恢复到执行前”。

## 远程运行安全

若当前唯一连接正依赖 SSH/Tailscale，工具默认阻止可能让自己断线的操作。请到目标机本地执行、准备第二条通道，或从另一台电脑完成外部验证。

当前 v0.2.0 未代码签名。遇到系统警告时请先核对 Release 页的 SHA-256，不建议关闭 SmartScreen 或安全软件。
